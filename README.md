# wardrobe

- 옷장관리 서비스
- [Dressroom] 웹스크래핑을 통한 구매한 옷 간편관리
- [Style] 오늘의 스타일과 어떤 아이템을 착용했는지 공유하는 서비스

## Tech-stack

- Frontend : Typescript + tailwind CSS + Next.js
- Backend : MongoDB
- Deploy : vercel
- State management : zustand, tanstack query
